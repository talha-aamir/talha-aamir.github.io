Neovim Config

github link

NOTE: unix/macos only. fzf doesn't work on windows, so you'll miss out on a major pro
With this config you get a coding experience similar but superior (in ways) to vs code. You get things like:

  • Gruvbox, my favorite color scheme
  • vim-fugitive: the greatest addition to vim, a git plugin. Conflict resolution has never been easier
  • COC: Conquer over completion. Allows code completion, syntax highlighting, easier code exploration, and much more...
  • fzf File search -> Search across file, search files across project, search in file
  • auto-update files on change
  • nerdtree: a file explorer
